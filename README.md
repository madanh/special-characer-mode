# special-characer-mode for emacs
Numerical row acts as if shift was pressed, numpad acts as usual

## Installation

Append contents of `.emacs` to your emacs init file (e.g. `~/.emacs`) and restart emacs

## Usage

```elisp
C-: (special-character-mode 1)
```
